# loan-predict：信贷违约风险预测项目
## 项目概述
  针对金融机构信贷业务的违约风险识别疼点，通过数据分析与机器学习建模，实现对借款人是否偿还贷款的预测，助力机构提前管控风险，优化信贷决策，降低坏账率。
## 业务背景
  在信贷业务中，银行或网贷等金融机构需要判断借款人员是否会按时还款，如果无法识别高违约风险的借款人，机构可能会面临巨额损失，本项目通过分析借款人特征与还款行为的关联，为机构提供风险预警工具。
## 数据说明
### 数据集规模：训练集共593994行、13列，测试集共254569行、13列
### 特征类型：
          数值型数据：年收入、债务收入比、信用分、贷款金额、利率
          分类型数据：性别、婚姻状态、教育程度、就业状态、贷款目的、等级
          标签：贷款是否偿还
### 数据质量：无缺失值、重复值，需进行特征筛选与预处理
## 技术栈
  数据处理：python、pandas、numpy
  可视化：seaborn、matplotlib
  特征工程：scikit-learn
  建模：Logistic Regression，svm，随机森林，Xgboost
## 数据分析流程
### 数据预处理
  1.特征筛选：删除无意义的id列
  2.数值型特征标准化：使用StandardScaler消除量纲差异，平衡特征对目标的影响
  3.分类型特征标准化：通过pandas.get_dummies或Onehotencoder将分类特征转化为数值格式
### 特征分析与可视化
  1.描述性统计：通过describe()分析数值特征的均值、标准差、分位数，把握数据分布
  2.相关性分析：利用热力图挖掘数值特征与标签的关联，发现信用分越高、利率越低，贷款偿还概率越高；债务收入比越高，违约风险越高。
  3.分组对比：对分类型特征进行统计，发现就业状态对还款行为影响显著，未偿还群体就业占比仅为偿还群体的50%，其中因素影响较小
  4.可视化验证：通过散点图，条形图直观的呈现核心特征与还款行为的关系。
